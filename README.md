## Instalación y Uso
### Requisitos
- Compilador de C++ (g++ recomendado)
- Make

### Compilación
Ejecutar el siguiente comando en la terminal:
```sh
make
```

### Ejecución
Para ejecutar el programa después de compilarlo:
```sh
make run
```
